\doxysubsection{Detailed Description}
Structure definition of ADC analog watchdog. 

\begin{DoxyNote}{Note}
The setting of these parameters by function HAL\+\_\+\+ADC\+\_\+\+Analog\+WDGConfig() is conditioned to ADC state. ADC state can be either\+:
\begin{DoxyItemize}
\item For all parameters except \textquotesingle{}High\+Threshold\textquotesingle{}, \textquotesingle{}Low\+Threshold\textquotesingle{}\+: ADC disabled or ADC enabled without conversion on going on ADC groups regular and injected.
\item For parameters \textquotesingle{}High\+Threshold\textquotesingle{}, \textquotesingle{}Low\+Threshold\textquotesingle{}\+: ADC enabled with conversion on going on regular and injected groups. 
\end{DoxyItemize}
\end{DoxyNote}
