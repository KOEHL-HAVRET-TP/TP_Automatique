◆ __HAL_RCC_ADC12_CONFIG

#define __HAL_RCC_ADC12_CONFIG (   __ADC12_CLKSOURCE__)     MODIFY_REG(RCC->CCIPR, RCC_CCIPR_ADC12SEL, (__ADC12_CLKSOURCE__))

Macro to configure the ADC12 interface clock.

Parameters
__ADC12_CLKSOURCE__specifies the ADC12 digital interface clock source. This parameter can be one of the following values:
  • RCC_ADC12CLKSOURCE_NONE No clock selected as ADC12 clock
  • RCC_ADC12CLKSOURCE_PLL PLL Clock selected as ADC12 clock
  • RCC_ADC12CLKSOURCE_SYSCLK System Clock selected as ADC12 clock
Return values
None
